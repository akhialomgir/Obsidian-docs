# 思路

1. 先尝试DP数组，设初值$dp=[1,2]$，其中各项为从第一阶开始的方法数。
2. 空间优化，用两个变量存储，考虑到范围为1开始，变量初始化为**第零阶和第一阶**，都为1。为了节约空间，直接使用参数$n$作为循环变量，结束条件设置为$n!==0$最为简洁，所以先累减再判断。

```js
//dp[i]=dp[i-2]+dp[i-1]
var climbStairs = function (n) {
    let p = 1, q = 1;
    while (--n) {
        [p, q] = [q, p + q];
    }
    return q;
};
```
